Activo total = Activos externos netos  +  Crédito al Gob. Nac.  +  Crédito al sist. financiero  +  Otros activos

Pasivo total = Base monetaria  +  Pasivos remunerados  +  Depósitos sector público  +  Oblig. con el exterior  +  Otros pasivos no monetarios  +  Patrimonio neto

Cualquier operación que sube el activo o baja un pasivo no monetario → expande la base monetaria. Lo inverso la contrae.
